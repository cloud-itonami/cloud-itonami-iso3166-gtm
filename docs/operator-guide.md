# Operator Guide

## First Deployment

1. Confirm the client's incorporation/legal-entity status is complete
   (route to `cloud-itonami-M6910` or local counsel first if not).
2. Register the client's intake: business type, target public
   function, declared contract value, prior filing history in
   Guatemala if any.
3. Run the advisor in read-only mode against GUATECOMPRAS
   (www.guatecompras.gt).
4. Compare the checklist against the client's current documentation,
   INCLUDING confirming whether the client's RGAE registration is mere
   Título II Inscripción or the materially heavier Título III
   Precalificación (Acuerdo Gubernativo 133-2024), and whether that
   tier actually covers the declared contract value under Guatemala's
   own statutory ladder (Ley de Contrataciones del Estado, Decreto
   57-92, Arts. 17/38/43 -- see README Checks item 3). A Cotización or
   Licitación Pública-tier engagement (> Q.90,000) needs
   Precalificación on file; a Compra Directa or Compra de Baja
   Cuantía-tier engagement (<= Q.90,000) does not.
5. Enable gated filing-draft assistance once the Market-Entry Compliance
   Governor contract is trusted; actual submission always requires human
   sign-off.

## Minimum Production Controls

- client-owned data store for business/tax registration documents
- clear provenance (official portal/regulation citation) for every
  requirement surfaced
- approval workflow for any portal registration or filing submission
- named referral relationship with Guatemalan-licensed counsel or a
  registered agent for anything beyond checklist/draft assistance
- monthly audit export

## Certification

Certified operators must prove data provenance, audit traceability, that
automated actions cannot bypass the Market-Entry Compliance Governor, and
a working referral relationship with Guatemalan-licensed counsel or a
registered agent for whatever licensed representation the law of
Guatemala requires for actual public-procurement filings.
